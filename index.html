IT-Adventures

A terminal-inspired cyberpunk RPG powered by JSON files and your command line.

Overview

IT-Adventures thrusts you into the neon-lit world of Nerdstown, where every corridor is lined with humming servers and every café serves “Espresso at 5 GHz.” You begin in your hacker den, armed only with a laptop and your wits. Explore diverse locations—from the open-space at TechCorp to the secret AI Lab—solve quizzes and fetch missions, collect upgrade items, earn reputation and credits, and become the ultimate sysadmin hero.

Key Features

  • Modular, JSON-driven world: rooms, items, missions and events are defined in simple JSON files.
  • Terminal interface: interact via familiar commands (e.g., go north, take, missions).
  • Dynamic puzzles and quizzes: test your knowledge of networking, Linux commands, scripting and open-source lore.
  • Collectible items: upgrade your gear with RAM sticks, soldering kits, power banks and more.
  • Non-player characters (NPCs): trade, challenge and unlock special rewards by engaging with in-world characters.
  • Event system: scheduled and on-enter triggers (DDoS alerts, power blackouts, retro quizzes) keep you on your toes.

Installation

  1. Clone the repository:
    git clone https://github.com/bocaletto-luca/IT-Adventures.git
  2. Enter the project directory:
    cd IT-Adventures
  3. Ensure you have Python 3.7+ installed.
  4. Install any dependencies (if provided):
    pip install -r requirements.txt

Usage

Launch the game with:

python3 game.py

In the terminal prompt, use commands like:

  • look: reprint the current room description
  • go <direction>: move north, south, east, west, etc.
  • take <item>: pick up an item
  • inventory: review collected items
  • missions: list and start available missions
  • use <item>: apply an item in the current room

Gameplay Details

As you travel through rooms, you’ll encounter mission prompts:

  • Quiz missions test your command-line knowledge (e.g., port numbers, grep flags, shebangs).
  • Fetch missions require you to locate and deliver items placed in specific rooms.
  • Event puzzles trigger on entry or on schedule (blackouts, DDoS alerts, retro quizzes).

Completing missions rewards you with experience points, in-game credits, open-source tokens and reputation. Level up and unlock 'sudo' privileges to access restricted areas.

Customization & Extensions

All game content lives in the data/ folder. You can:

  • Add or modify rooms.json, items.json, missions.json and npcs.json.
  • Define new events in events.json and hook them into rooms with the event field.
  • Tweak rewards, difficulty and prerequisites directly in JSON to tailor your own adventure.
